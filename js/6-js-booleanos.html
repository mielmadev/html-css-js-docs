Booleanos en JavaScript

Operaciones básicas con booleanos

¿Qué es un booleano?

Los booleanos son un tipo de dato que solo puede tener dos valores: true (verdadero) o false (falso). Se utilizan para representar estados binarios, como encendido/apagado, sí/no, verdadero/falso, etc.

En programación puedes verificar que algo sea mayor, menor o igual, con booleanos se puede hacer comparaciones.

Los booleanos son muy útiles en estructuras de control como condicionales y bucles, donde se evalúa una condición para determinar el flujo del programa.

Negación

El operador de negación (!) invierte el valor de un booleano:

var rebajas = true;
console.log(rebajas);   // true
console.log(!rebajas);  // false

var activado = false;
console.log(activado);	// false
console.log(!activado);	// true

Operadores lógicos

Los operadores lógicos permiten combinar múltiples condiciones booleanas:

  • AND (&&): Devuelve true solo si ambas condiciones son verdaderas
  • OR (||): Devuelve true si al menos una condición es verdadera

var valor1 = true;
var valor2 = true;
var valor3 = false;
var valor4 = false;

// AND - &&
var resultado1 = valor1 && valor2; // true
document.write("¿valor1 y valor2 son verdaderos? " + resultado1 + "
");
var resultado2 = valor1 && valor3; // false
document.write("¿valor1 y valor3 son verdaderos? " + resultado2 + "
");
var resultado3 = valor3 && valor4; // false
document.write("¿valor3 y valor4 son verdaderos? " + resultado3 + "
");

// OR - ||
var resultado4 = valor1 || valor2; // true
document.write("¿valor1 o valor2 es verdadero? " + resultado4 + "
");
var resultado5 = valor1 || valor3; // true
document.write("¿valor1 o valor3 es verdadero? " + resultado5 + "
");
var resultado6 = valor3 || valor4; // false
document.write("¿valor3 o valor4 es verdadero? " + resultado6 + "
");

Operadores relacionales

Los operadores relacionales se utilizan para comparar valores y devuelven un resultado booleano:

Operadores de comparación

  • > - Mayor que
  • < - Menor que
  • >= - Mayor o igual que
  • <= - Menor o igual que
  • != - Diferente (no igual)

Operadores de igualdad

Para comparar valores se usan == y ===:

  • El operador == compara los valores de dos variables, pero no su tipo. Por ejemplo, 5 == '5' es verdadero porque ambos tienen el mismo valor, aunque uno es un número y el otro es una cadena de texto.
  • El operador === compara tanto el valor como el tipo de las variables. Por ejemplo, 5 === '5' es falso porque uno es un número y el otro es una cadena de texto, a pesar de que tienen el mismo valor.

Recomendación: Es recomendable usar siempre === para evitar resultados inesperados debido a la coerción de tipos que ocurre con ==.

Buenas prácticas y notas importantes

  • Se recomienda poner las comparaciones entre paréntesis, ya que más adelante será obligatorio al usarlas dentro de otras estructuras.
  • No se pueden comparar arrays directamente con == o ===, ya que estos operadores comparan referencias, no el contenido. Dos arrays con el mismo contenido serán considerados diferentes si no son el mismo objeto.
var numero1 = 10;
var numero2 = 50;
var numero3 = 50;

var rdo1 = numero1 > numero2;   // false
var rdo2 = numero1 < numero2;   // true

var rdo3 = numero1 >= numero2;  // false
var rdo4 = numero1 <= numero2;  // true

var rdo5 = numero2 <= numero3;  // true
var rdo6 = numero2 >= numero3;  // true

var rdo7 = numero1 == numero2;  // false
var rdo8 = numero2 == numero3;  // true
var rdo9 = numero1 != numero3;  // true
var rdo10 = numero2 != numero3; // false

var texto1 = 'día';
var texto2 = 'noche';
var texto3 = 'día';
var texto4 = 'dia';
var texto5 = 'Día';

var rdo7 = texto1 == texto2;   // false
var rdo8 = texto1 == texto3;   // true
var rdo8a = texto1 == texto4;  // false
var rdo8b = texto1 == texto5;  // false

var rdo9 = texto1 != texto2;   // true
var rdo10 = texto1 != texto3;  // false
var rdo10b = texto1 != texto4; // true

// Comparación de tipos diferentes
var n1 = 100;
var n2 = '100';

var r1 = n1 == n2;  // true  (solo compara valor)
var r2 = n1 === n2; // false (compara valor y tipo)
var r3 = n1 !== n2; // true  (diferentes tipos)
