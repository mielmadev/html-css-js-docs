Librerías de JavaScript

JavaScript cuenta con una amplia variedad de librerías que facilitan el desarrollo y la manipulación de datos. Algunas de las más populares incluyen:

  • jQuery: Simplifica la manipulación del DOM y la gestión de eventos. Se utilizaba porque a JavaScript "vanilla" le faltaban utilidades para trabajar fácilmente con el DOM y los eventos. Actualmente, la mayoría de lo que hacía jQuery se puede hacer con JavaScript moderno (vanilla) o incluso con CSS.
  • Lodash: Simplifica el trabajo con arrays, objetos y funciones. Se utiliza porque JavaScript "vanilla" no tenía métodos avanzados para manipular datos de forma sencilla. Hoy en día, muchas de sus utilidades están disponibles en JavaScript moderno, pero Lodash sigue siendo útil para compatibilidad y operaciones complejas.
  • Axios: Facilita las solicitudes HTTP y la gestión de promesas. Se utiliza porque JavaScript "vanilla" no tenía una forma sencilla y uniforme de hacer peticiones AJAX y manejar respuestas en todos los navegadores. Actualmente, fetch es la alternativa moderna, pero Axios sigue siendo útil por su compatibilidad y características avanzadas.

La librería que usa Ainhoa para los colores del código en el HTML es highlight.js.

La mejor manera de elegir una librería es que no tenga dependencias, que esté hecha en vanilla JavaScript, que tenga buena compatibilidad y que el HTML que haya que poner sea estándar.
