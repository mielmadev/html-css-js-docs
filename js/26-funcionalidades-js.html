for...of

Es similar a for...in, pero recorre valores en vez de claves.

El bucle for...of permite iterar sobre objetos iterables como arrays, strings, etc.

const array = [1, 2, 3];
for (const elemento of array) {
  console.log(elemento);
}

forEach

Solo funciona con arrays, es obligatorio llamarlo desde un array. Nació porque en otra librería existía ese concepto.

forEach es un método que se llama desde el array, no es un bucle como tal. Utiliza una función anónima que puede tener hasta tres parámetros: valor (elemento), índice y el array completo.

El método forEach permite ejecutar una función en cada elemento de un array.

const array = [1, 2, 3];
array.forEach(function(elemento) {
  console.log(elemento);
});

Comparación de los 4 tipos de bucles

Bucle Uso principal Sobre qué itera Acceso a índice Permite break/continue Notas
for Control total de la iteración Arrays, strings, cualquier iterable Sí (i) Sí Ideal para recorrer arrays con índice o cuando necesitas saltos/condiciones
for...in Recorrer propiedades enumerables Objetos, arrays (índices como string) Sí (clave) Sí No recomendado para arrays (el orden no está garantizado)
for...of Recorrer valores de iterables Arrays, strings, Map, Set No (solo valor) Sí No funciona en objetos normales
forEach Ejecutar función en cada elemento de un array Arrays Sí (como segundo parámetro) No No se puede usar break/continue, solo return dentro de la función

Cadenas

Las cadenas de texto en JavaScript son objetos inmutables que representan una secuencia de caracteres.

Para poder insertar variables dentro de una cadena y evitar problemas con comillas y espacios, se usa la sintaxis ${} dentro de las comillas invertidas o backticks (`aquí va el resultado ${resultadoFuncion}`).

Condicionales

El operador ternario es una forma concisa de escribir una estructura condicional if...else en una sola línea. Solo devuelve un valor, no sirve para dos líneas, es para un único if...else.

condición ? expresiónSiVerdadero : expresiónSiFalso;

Ejemplo:

const nombreVariable = condiciónIfElse ? valorSiVerdadero : valorSiFalso;
              console.log(nombreVariable);
        
const edad = 18;
const esMayorDeEdad = edad >= 18 ? "Sí" : "No";
console.log(esMayorDeEdad);

Expresión de función

Una expresión de función es una forma de definir una función en JavaScript. A diferencia de las declaraciones de función, que se definen con la palabra clave function, las expresiones de función se pueden asignar a variables y usar como valores.

// Declaración de función
function saludar(nombre) {
  return "Hola, " + nombre + "!";
}
console.log(saludar("MielmaDev")); // Hola, MielmaDev!

Funciones flecha

Las funciones flecha (arrow functions) son una forma más concisa de escribir funciones en JavaScript. Se introdujeron en ES6 y tienen una sintaxis más corta que las funciones tradicionales.

Se ponen paréntesis si no hay parámetros o si hay varios; cuando solo hay un parámetro, no se ponen paréntesis.

Agregar función con parámetros

// Función tradicional
function suma(a, b) {
  return a + b;
}
console.log(suma(5, 3)); // 8

// Función flecha
const suma = (a, b) => a + b;
console.log(suma(5, 3)); // 8

Parámetros por defecto

Los parámetros por defecto permiten asignar un valor predeterminado a un parámetro de una función en caso de que no se proporcione un valor al llamar a la función.

Se puede poner un valor por defecto a los parámetros que se especifican, por ejemplo:

function saludar(nombre = "Mundo") { return "Hola, " + nombre + "!"; }
        saludar(); // "Hola, Mundo!"
        saludar("MielmaDev"); // "Hola, MielmaDev!"

Parámetros rest

Los parámetros rest permiten representar un número indefinido de argumentos como un array. Se definen utilizando tres puntos (...) antes del nombre del parámetro. Al menos un parámetro es obligatorio.

Si fueran nombres, hay que recorrerlos como un array.

function sumar(...numeros) {
            let total = 0;
            for (let i = 0; i < numeros.length; i++) {
              total += numeros[i];
            }
            return total;
            }
            console.log(sumar(1, 2, 3)); // 6
            console.log(sumar(5, 10, 15, 20)); // 50

map

El método map() crea un nuevo array con los resultados de la función proporcionada aplicada a cada elemento del array original. Si quieres transformar los datos de un array a otro formato, puedes usar map().

const numeros = [1, 2, 3, 4, 5];
const dobles = numeros.map(num => num * 2);
console.log(dobles); // [2, 4, 6, 8, 10]

filter

El método filter() crea un nuevo array con todos los elementos que cumplan la condición implementada por la función proporcionada. Si quieres filtrar elementos de un array según algún criterio, puedes usar filter().

const numeros = [1, 2, 3, 4, 5];
const pares = numeros.filter(num => num % 2 === 0);
console.log(pares); // [2, 4]

reduce

El método reduce() aplica una función contra un acumulador y cada elemento del array (de izquierda a derecha) para reducirlo a un solo valor. Si quieres realizar una acumulación o una operación que implique todos los elementos de un array, puedes usar reduce().

const numeros = [1, 2, 3, 4, 5];
const suma = numeros.reduce((acumulador, actual) => acumulador + actual, 0);
console.log(suma); // 15
