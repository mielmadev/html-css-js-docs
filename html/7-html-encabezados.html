7. Encabezados

Mediante los títulos (también llamados encabezados), estructuraremos la información de la página en base a los contenidos y sus jerarquías.

Niveles

Disponemos de 6 elementos de título (h1, h2, h3, h4, h5, h6) donde h1 es el título más importante. Por ejemplo: 

h1  Título principal de la página
│
├─ h2  Sección 1
│   │
│   ├─ h3  Subsección 1.1
│   │   │
│   │   └─ h4  Detalle 1.1.1
│   │
│   └─ h3  Subsección 1.2
│
├─ h2  Sección 2
│   │
│   ├─ h3  Subsección 2.1
│   └─ h3  Subsección 2.2
│       │
│       └─ h4  Detalle 2.2.1
│
└─ h2  Sección 3
    │
    ├─ h3  Subsección 3.1
    └─ h3  Subsección 3.2

Normas

Como norma básica recomendada, en la página vamos a tener un único elemento h1 y a partir de ahí iremos construyendo el esquema de títulos/encabezados de los contenidos de la página.

Cada elemento título tendrá su correspondiente etiqueta de apertura y de cierre.

No debemos utilizar un h sin haber utilizado previamente su h padre. Por ejemplo, no deberíamos utilizar h2 en una página si previamente no hemos utilizado h1.

Ejemplo correcto

	h1 Servicios
	├─ h2 Catering
	├─ h2 Preparación oposiciones
	│  ├─ h3 Ayudante de Chef
	│  └─ h3 Ayudante de cocina
	└─ h2 Formación privada


<h1>Servicios</h1>
<p>Empresa ACME le proporciona los siguientes servicios:</p>

<h2>Catering</h2>
<p>Por un módico precio, puede disfrutar de la mejor y más variada
comida del mundo…</p>

<h2>Preparación oposiciones</h2>
<p>Nuestro personal estará encantado en ayudarle a preparar las
siguientes oposiciones:</p>

<h3>Ayudante de Chef</h3>
<p>bla, bla, bla</p>

<h3>Ayudante de cocina</h3>
<p>bla, bla, bla</p>

<h2>Formación privada</h2>
<p>Si lo desea, podemos enseñarle a preparar usted mismo los exquisitos
platos...</p>

Ejemplo incorrecto

	│
	└─ h2  Servicios
│
└─ h1  Catering
	│
	└─ h2  Preparación oposiciones
		│
		├─ h3  Ayudante de Chef
		│
		└─ h3  Ayudante de cocina
			│
			└─ h4  Formación privada


<h2>Servicios</h2>
