8. Listas

Una lista es un conjunto/agrupación de elementos que pueden tener o no una relación entre sí.

Vamos a usar listas para implementar los menús de navegación de la página web, el camino de migas, la paginación de unos resultados de búsqueda, etc... Bastará con que utilicemos CSS para cambiar el aspecto visual.

En HTML existen 3 tipos de listas: desordenadas, ordenadas y de definición.

Listas desordenadas

Los elementos de esta lista no tienen un orden determinado y a nivel visual, cada elemento de la lista aparece con una viñeta.

Para crear una lista desordenada, primero usamos el elemento ul (unorderer list) con su correspondiente etiqueta de apertura y cierre y dentro añadiremos los elementos de la lista, cada uno de ellos con la etiqueta li (list item).

See the Pen HTML - Lista desordenada by Antxoa (@antxoa) on CodePen. 


<h1>PAN DE LEMBAS</h1>

<h2>Ingredientes</h2>
<ul>
	<li>50g de harina regular</li>
	<li>50g de harina integral</li>
	<li>4 cucharadas de miel</li>
	<li>1 pizca de levadura química o de panadero</li>
	<li>1 pizca de sal</li>
	<li>Un puñadito de nueces</li>
	<li>3 cucharadas de aceite</li>
	<li>70g de almendra molida</li>
	<li>1 huevo</li>
	<li>La rayadura de un limón</li>
	<li>El puñetazo de un ser supremo</li>
</ul>

Fuente: ℗ Pan de lembas o pan élfico (Lord Of The Rings) | Superpilopi

Listas ordenadas

Los elementos de la lista guardan un orden específico y numeran sus elementos.

La forma de crear esta lista es igual que la lista desordenada, sólo cambia su etiqueta principal, que será ol (ordered list):

See the Pen HTML - Lista ordenada by Antxoa (@antxoa) on CodePen. 


<h1>PAN DE LEMBAS</h1>

<h2>Preparación</h2>

<ol>
	<li>Añadir en un bol todos los ingredientes, excepto las nueces.</li>
	<li>Añadir nueces troceadas y arrojarlas con furia a la mezcla.</li>
	<li>Remover todo con cuchara (o garra mortal) hasta obtener una bola pegajosa y marrón.</li>
	<li>Pasar la masa a superficie enharinada, amasar y formar una bola. Dejar reposar 10 min.</li>
	<li>Estirar con rodillo hasta 1-2 cm de grosor.</li>
	<li>Recortar con plantilla cuadrada y guardar recortes para otro pan.</li>
	<li>Marcar línea diagonal de pico a pico con palillo u objeto fino.</li>
	<li>Hornear sobre papel a 200 °C (arriba y abajo) 20 min o hasta dorar.</li>
	<li>Dejar enfriar y disfrutar.</li>
</ol>

Mediante CSS podremos modificar la apariencia de las listas: cambiar las viñetas, el tipo de numeración (números romanos, letras...) y la orientación de las listas (vertical u horizontal).

Listas anidadas

Podemos anidar una lista dentro de otra y así generar diferentes niveles de listas.

La sublista siempre será hija de un elemento li, ya que ul y ol sólo pueden contener elementos li.

See the Pen HTML - Listas anidadas by Antxoa (@antxoa) on CodePen. 


Correcto:

<ul>
	<li>
		Trilogía de El señor de los anillos
		<ol>
			<li>La comunidad del anillo</li>
			<li>Las dos torres</li>
			<li>El retorno del rey</li>
		</ol>
	</li>
	<li>El Silmarillión</li>
	<li>El Hobbit</li>
</ul>


Incorrecto:

<ul>
	<li>Trilogía de El señor de los anillos</li>
	<ol>
		<li>La comunidad del anillo</li>
		<li>Las dos torres</li>
		<li>El retorno del rey</li>
	</ol>
	<li>El Silmarillión</li>
	<li>El Hobbit</li>
</ul>

Listas de definición

A diferencia de las listas anteriores, las listas de definición, representadas por el elemento dl, tienen un par de elementos por cada item: un término, dt, y su definición, dd.

Es posible que un término pueda tener varias definiciones:

See the Pen HTML - Lista de definición by Antxoa (@antxoa) on CodePen. 


<h1>El señor de los anillos: La comunidad del anillo</h1>

<dl>
	<dt>Año:</dt>
	<dd>2001</dd>

	<dt>Director:</dt>
	<dd>Peter Jackson</dd>

	<dt>Música:</dt>
	<dd>Howard Shore</dd>
