4. Texto

Párrafos

Mediante los párrafos, estructuraremos la mayor parte de los contenidos web: 

<p>este es un párrafo</p>

<p>este es otro párrafo</p>

Saltos de línea

Se puede dar el caso en el que necesitemos saltar de línea dentro de un párrafo, por ejemplo para estructurar las estrofas de una canción, poema...

Para ello utilizaremos el elemento br cuya etiqueta es autoconclusiva: 

<p>
	Imagine there’s no heaven<br>
	It’s easy if you try
</p>

No debemos utilizar dos elementos br seguidos sólo para simular un párrafo. Lo tendremos que realizar con propiedades CSS para aumentar o disminuir la separación entre párrafos.

Énfasis

Cuando queremos destacar un texto, solemos mostrarlo en negrita y, al leerlo, lo hacemos con otro tono de voz.

Antiguamente utilizábamos los elementos b e i (bold e italic) para tal efecto,pero eran simplemente etiquetas de diseño. Cuando un navegador parlante leía ese texto, no cambiaba la entonación, porque para él, no tenía ningún significado especial. No eran elementos semánticos sino visuales. En la actualidad esos dos elementos tienen otros propósitos que veremos más adelante.

Por tanto, para indicar si un texto es importante o muy importante, disponemos de 2 elementos semánticos: em y strong: 

<p>
	<strong>Aviso:</strong> No olvide utilizar un <em>DNI válido</em>.
</p>

El estilo por defecto de estos elementos (cómo se muestra en los navegadores) es la negrita para strong y la itálica (cursiva) para em pero este formato de visualización lo podremos cambiar más adelante mediante CSS.

Citas

Para hacer referencia a un texto a modo de cita, disponemos de 2 elementos para ello:

Bloque de cita

Cuando queremos mostrar un bloque de texto citado textualmente, debemos utilizar este elemento: 

<blockquote>
	<p>En un lugar de la Mancha, de cuyo nombre no quiero acordarme, no ha mucho tiempo que vivía un hidalgo de los de lanza en astillero, adarga antigua, rocín flaco y galgo corredor.</p>
	<p>Una olla de algo más vaca que carnero, salpicón las más noches, duelos y quebrantos los sábados, lentejas los viernes, algún palomino de añadidura los domingos, consumían las tres partes de su hacienda.</p>
</blockquote>

No se debe utilizar este elemento simplemente para utilizar la sangría que proporcionan por defecto, si no estamos citando un texto.

Citas en línea

Para citas en línea y cortas, utilizaremos el elemento q para el texto y cite para el autor. Los navegadores suelen mostrar cite en cursiva pero este formato se puede modificar con CSS. 

<p>Como dijo <cite>Hamlet</cite>: <q>ser o no ser, esa es la cuestión</q>.</p>

Abreviaturas

Para marcar cualquier tipo de abreviatura, sigla o acrónimo, se utiliza el elemento abbr.

Si queremos expandir su significado, utilizaremos el atributo title, con el significado correspondiente. 

<abbr title="Unión Europea">UE</abbr>

El atributo title se puede utilizar en la práctica totalidad de los elementos HTML (enlaces, campos de formulario...) para complementar la información del elemento al que acompaña. El valor será el texto que nosotros determinemos, totalmente libre.

Importante no confundir el atributo title del elemento/etiqueta <title></title> del head. El nombre es el mismo pero son dos elementos totalmente diferentes.
