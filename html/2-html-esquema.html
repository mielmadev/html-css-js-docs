2. Esquema básico

Ahora que conocemos la mecánica de las etiquetas y los atributos, vamos a empezar a crear nuestro primer código web.

html: Elemento/Etiqueta principal

El elemento principal es html que se representa con las etiquetas de apertura <html> y de cierre </html>. Entre la apertura y el cierre se insertarán el resto de elementos. 


<html>

</html>

Además, para indicar explícitamente al navegador que estamos haciendo una página html, indicamos con una línea estándar, al principio de todo de nuestro código, el estándar que estamos escribiendo.

Esta línea en realidad es una etiqueta especial que no tiene etiqueta de cierre. Se tiene que poner tal cual. 


<!DOCTYPE html>
<html>

</html>

lang: Idioma principal

En la etiqueta <html> vamos a añadir un atributo que sirve para indicar el idioma en el que vamos a tener la información de la página web. Esto servirá posteriormente para que los buscadores puedan indexar mejor el idioma de los contenidos, que se puedan hacer traducciones en el propio navegador, etc.

Este atributo es lang y el valor tiene que corresponder con el código ISO 639 de 2 caracteres: 


<!DOCTYPE html>
<html lang="es">

</html>

Adicionalmente también se puede establecer el código ISO 3166 correspondiente a idioma y región: 


<html lang="es">   <html lang="es-ES">   <html lang="es-MX">

<html lang="en">   <html lang="en-UK">   <html lang="en-US">   <html lang="en-AU">

Este atributo no es obligatorio pero es esencial para el buen funcionamiento de nuestra página web, por lo tanto, lo vamos a poner siempre.

head: el encabezado (invisible)

El elemento html va a contener obligatoriamente 2 elementos más. Estos 2 elementos serán sus hijos y no se puede prescindir de ellos, ni añadir otros. Son dos elementos únicos.

El primer elemento es head, representado por la etiqueta <head>. Esta etiqueta va inmediatamente después de abrir <html>, es decir, es su primer hijo: 


<!DOCTYPE html>
<html lang="es">

    <head>

    </head>

</html>

Dentro de head se ubica una gran cantidad de información técnica que no será visible en el navegador, pero sí se necesitará a nivel operativo, por ejemplo: asociación con código CSS, JavaScript, iconos para marcadores, codificación de caracteres, información para buscadores, redes sociales, etc.

A medida que avance el curso iremos añadiendo más información a este elemento, pero por el momento nos vamos a centrar en lo imprescindible: la codificación de caracteres y el título interno de la página.

Codificación de caracteres

Para que el navegador sepa qué codificación de caracteres debe utilizar a la hora de mostrar los textos, es necesario indicar explícitamente esta información. No es lo mismo tener una página escriba en inglés que una en chino, puesto que los caracteres son totalmente diferentes.

Existen múltiples codificaciones y para los idiomas occidentales vamos a utilizar utf-8 (en el pasado se utilizaba ISO-8859-1) y para indicarlo en nuestra página web vamos a hacer uso del elemento meta y el atributo charset: 


<!DOCTYPE html>
<html lang="es">

    <head>

        <meta charset="utf-8">

    </head>

</html>

La etiqueta meta se usa exclusivamente dentro de head y tiene múltiples usos. Es por ello que podemos tener varias meta, cada una con diferentes atributos y valores.

Es una etiqueta autoconclusiva, es decir, no va a existir etiqueta de cierre, por lo que no podemos inventarnos la etiqueta de cierre: 


<!-- esto es incorrecto -->
<meta charset="utf-8">
</meta>

title: título interno

Cuando navegamos, en las pestañas siempre aparece un icono y un texto. De tal manera de con la combinación de ambos podemos diferenciar una pestaña de otra. Y si añadimos la página a nuestros marcadores/favoritos también aparecerá esa información.

Ese texto que aparece en la pestaña es el elemento title de la página. No tiene por qué coincidir con el que aparecerá en nuestra pantalla de contenidos, es un título para visualizar en la pestaña y marcadores.

Este elemento es único y va ubicado dentro de head, preferiblemente después de la codificación de caracteres. Si va antes, es posible que si el texto que pongamos tiene caracteres especiales no se visualicen correctamente puesto que aún no le hemos dicho al navegador cuál es la codificación de caracteres: 


<!DOCTYPE html>
<html lang="es">

    <head>

        <meta charset="utf-8">
        <title>Mi primera página</title>

    </head>

</html>

body: el cuerpo (visible)

El segundo elemento hijo de html es body, representado por la etiqueta <body>. Esta etiqueta va inmediatamente después de cerrar <head>, es decir, es su hermano y el segundo hijo de html: 


<!DOCTYPE html>
<html lang="es">

    <head>

        <meta charset="utf-8">
        <title>Mi primera página</title>

    </head>

    <body>

        ... El contenido visible ...

    </body>

</html>

Dentro de body se ubican todas las etiquetas que necesitamos para elaborar la página web, cuyo resultado se visualizará en el navegador.
