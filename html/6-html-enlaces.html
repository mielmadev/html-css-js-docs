6. Enlaces

Los enlaces, o también llamados vínculos, hiperenlaces... son la piedra angular de las páginas web.

Estructura general

Los enlaces se implementan mediante el elemento a y se componen, básicamente, de dos partes: ruta (atributo href) y texto del enlace.

Si la página que vamos a enlazar se encuentra en la misma carpeta que la página en la que estamos, bastará con escribir el nombre del fichero.

Si está alojada en una subcarpeta, entonces utilizamos una barra / para indicar el camino y si se encuentra en una carpeta de un nivel superior, utilizamos dos puntos y una barra ../ para subir de nivel: 


<a href="ruta">texto del enlace</a>

<a href="mipagina.html">página personal</a>

<a href="personal/mipagina.html">mi página</a> (en subdirectorio)

<a href="../mipagina.html">mi página</a> (en nivel superior)

Podemos subir o bajar de nivel las veces que sea necesario:


<a href="personal/io/mio/mipagina.html">mi página</a>

<a href="../../../mipagina.html">mi página</a>

Enlace a una página externa:

Los enlaces externos necesitan llevar una estructura estándar en cuanto a su ruta:

  • Empiezan por http:// o https://
  • Después puede ir www o un subdominio
  • Finaliza con el nombre del domino

Incorrecto:
<a href="google.es">Google</a>
<a href="www.google.es">Google</a>
<a href="http:www.google.es">Google</a>
<a href="https://www.play.google.com/">Google Play</a>


Correcto:
<a href="http://www.google.es">Google</a>
<a href="https://www.google.es">Google</a>
<a href="https://play.google.com/">Google Play</a>


Correcto, si se ha configurado el dominio para permitirlo:
<a href="http://google.es">Google</a>
<a href="https://google.es">Google</a>

Complementar información del enlace

Podemos utilizar el atributo title para ofrecer una pequeña descripción de la página a la que vamos a enlazar. No tendrá mucho sentido si el texto del título es idéntico o similar al texto del enlace.

Este atributo es opcional y sólo debe usarse sabiendo muy bien el por qué y en caso de ofrecer información adicional 


Incorrecto:
<a href="http://www.yahoo.com" title="Yahoo">Yahoo</a>

Correcto:
<a href="http://www.yahoo.com" title="Yahoo: noticias, e-mail, fotos, grupos, etc...">Yahoo</a>

Enlaces en nueva ventana/pestaña

Durante años, la costumbre más extendida para abrir los enlaces externos en una nueva ventana era utilizar el atributo target="_blank" en el enlace.

Sin embargo, en la actualidad, esta práctica está desaconsejada, puesto que los navegadores utilizan mecanismos para no abrir en una nueva ventana (sino en una nueva pestaña) o incluso pueden bloquer esta orden.

Por lo tanto utilizaremos soluciones javascript (por ejemplo, marcando el enlace como externo con el atributo rel="external" y aplicando una función javascript sobre este atributo) que veremos a su debido tiempo.

Enlaces dentro de una página

Para enlaces dentro de la misma página o dentro de otra página, utilizamos lo que se denomina una referencia. En el elemento donde queremos saltar, insertamos una referencia mediante el atributo id y después implementamos un enlace que lleve a ese elemento.

Como el enlace no apunta ni a un archivo ni a una página externa, sino que apunta a una referencia, será necesario empezar la ruta del enlace con una almohadilla #: 


<a href="#cancion2">Canción 2</a>

<a href="noticias.html#referencia1">Referencia</a>

...........

<p id="cancion2">Esta es la Canción 2</p>

El valor del atributo id es libre pero tiene que seguir unas normas:

  • Tiene que ser único en la página, no puede haber duplicados
  • No puede empezar con un número pero sí puede contenerlo (cancion8 es correcto, pero 8canción es incorrecto)
  • No puede contener espacios
  • No puede contener caracteres especiales, acentuados...

Otros enlaces

Podemos hacer enlaces especiales a una dirección de e-mail, un teléfono, whatsapp, etc... 


<a href="mailto:hola@acme.es">envía a un e-mail</a>

<a href="tel:944 444 444">944 444 444</a>

<a href="https://wa.me/34944444444?text=hola">Envía por Whatsapp</a>

<a href="https://api.whatsapp.com/send?phone=+34944444444&text=hola">Envía por Whatsapp</a>


  • envía a un e-mail
  • 944 444 444
  • Envía por Whatsapp
  • Envía por Whatsapp

Idioma de destino de un enlace

Si un enlace apunta a una página en un idioma diferente al que estamos, podemos indicar ese lenguaje mediante el atributo hreflang: 


<a href="http://www.nasa.gov" hreflang="en">NASA</a>

<a href="http://www.paris.fr" hreflang="fr">Ayuntamiento de París</a>

Este atributo es similar al atributo lang, en cuanto a los códigos de idioma/región.

Imágenes como enlace

Es posible utilizar una imagen como enlace, basta con reemplazar el texto del enlace con la imagen: 


<a href="index.html">mapa web</a>

<a href="index.html">
	<img src="mapa.gif" alt="mapa web">
</a>
