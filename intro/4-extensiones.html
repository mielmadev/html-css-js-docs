4. Extensiones

Para mejorar las prestaciones de los navegadores de cara al desarrollo, vamos a instalar un par de extensiones, que utilizaremos a su debido tiempo:

  • Web Developer
  • Colorzilla

A lo largo del curso se recomendarán otras extensiones para otras pruebas puntuales.

 

También vamos a utilizar mucho las siguientes herramientas propias de los navegadores:

  • Vista de Código fuente: Ctrol + u
  • Consola de desarrollo: Ctrol + Mayus. + i

Referencias

  • Web Developer: Chrome y Firefox
  • Colorzilla: Chrome y Firefox
