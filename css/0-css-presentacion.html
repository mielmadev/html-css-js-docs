0. Presentación

CSS (Cascading Style Sheets - Hojas de Estilo en Cascada) es un lenguaje de hoja de estilo creado por el W3C que permite añadir estilos (tipografías, tamaños, espaciado, color...) y maquetación a los documentos Web.

Separando el estilo de presentación del contenido de los documentos, CSS simplifica la creación y mantenimiento de los sitios Web.

Estado actual de la especificación CSS

CSS está dividido en diferentes módulos (colores, bordes, posicionamiento...) y en cada módulo, los editores trabajan a ritmos diferentes, por lo que algunos módulos han finalizado su desarrollo y otros acaban de empezar a trabajar.

Soporte actual

Un motor de renderizado es el encargado de traducir el código de una página web (HTML, CSS, JavaScript) en los elementos visuales que podemos ver y con los que podremos interactuar en nuestro navegador.

Cada navegador utiliza un motor de renderizado concreto y va adoptando las nuevas propiedades a su ritmo. Hasta hace unos años, mientras los navegadores probaban que la propiedad funcionase bien, solían implementar una versión beta de dicha propiedad. Esta implementación se realizaba utilizando unos prefijos específicos (en función de cada motor de renderizado) delante de la propiedad CSS a utilizar (por ejemplo -moz-nombrePropiedad, -webkit-nombrePropiedad...) y en algunos ejemplos y páginas de referencia aparecerán.

En la actualidad, los navegadores han modificado este método por otro más óptimo. En vez de utilizar prefijos, utilizamos directamente la propiedad estándar y en la configuración del navegador habilitamos la característica experimental.
