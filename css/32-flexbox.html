32. Flexbox

¿Qué es Flexbox?

Flexbox consiste en un conjunto de propiedades creadas ex proceso para maquetar los contenidos. Nos va a permitir maquetar contenidos en diferentes columnas, hacer alineaciones verticales perfectas para bloques, distribuir contenido uniformemente, igualas en altura todo un conjunto de elementos etc.

a diferencia de las propiedades CSS que hemos visto hasta ahora, el uso deFlexbox consta de dos fases

  1. Primero se aplican las propiedades generales al elemento contenedor(al padre) de los elementos que queremos maquetar. Muchas veces con esto es suficiente es suficiente
  2. Después, podemos poner ciertas propiedades a los elementos hijo, que modificarán ciertas órdenes del padre

Propiedades para el Elemento Contenedor (padre)

Para empezar a utilizar las propiedades tenemos que indicarle al elemento padre que va a tener una visualización de tipo flexbox

Crecimiento

La propiedad flex-grow permite aplicarle al elemento la habilidad de crecer en el espacio libre.

Se establece un valor sin unidad de medida que en realidad representa la proporción en la que debe crecer el elemento. Por defecto, todos los elementos tienen un valor de 0, que indica que no tienen esta la habilidad de crecer. Si insertamos cualquier otro valor en esta propiedad, el elemento crecerá (hasta ocupar todo el espacio libre).

Si hay más de un elemento con la propiedad flex-grow, los elementos que tengan el mismo valor en la propiedad se repartirán el espacio libre entre ellos, para que crezcan al mismo tamaño. Para los elementos que tengan valores diferentes, los valores representarán partes del espacio sobrante.

Al aplicar esta propiedad, la anchura y/o altura que se hayan establecido a los elementos se ignoran, para aplicar el crecimiento establecido en la propiedad.

Tabla de flex-grow

Valor Descripción
flex-grow: 0; No crece, ocupa solo el espacio necesario.
flex-grow: 1; Crece para ocupar el espacio libre disponible.
flex-grow: n; Crece en proporción al valor asignado respecto a otros elementos.

Decrecimiento

La propiedad flex-shrink es la inversa de la propiedad anterior y estable cuánto se debe comprimir (decrecer) un elemento cuanto no queda espacio libre.

Por defecto, todos los elementos tienen un valor de 1. Si insertamos cualquier otro valor en esta propiedad, el elemento se comprimirá en proporción más que los otros.

Tabla de flex-shrink

Valor Descripción
flex-shrink: 1; Valor por defecto, el elemento puede reducirse si es necesario.
flex-shrink: 0; El elemento no se reduce, mantiene su tamaño.
flex-shrink: n; El elemento se reduce en proporción al valor asignado respecto a otros elementos.

Tamaño Base

La propiedad flex-basis establece un ancho base inicial para un elemento. El valor por defecto es auto y en este caso respeta el ancho establecido en el elemento (si lo hay). Con cualquier otro valor (px, %, ...) no respeta el ancho establecido y aplicará el valor indicado en flex-basis:

Tabla de ejemplos de flex-basis

Selector Propiedad ¿Respeta el width?
.item:nth-child(1) flex-basis: 175px; No
.item:nth-child(2) flex-basis: auto; Sí
.item:nth-child(3) flex-basis: 125px; No
.item:nth-child(4) flex-basis: auto; Sí
.item:nth-child(5) flex-basis: 150px; No
auto flex-basis: auto; Sí

Combinación de propiedades

Podemos combinar los valores de las propiedades flex-grow, flex-shrink y flex-basis en una única propiedad flex:

Podemos combinar los 3 valores para lograr algunos ajustes:

  • flex: auto; Equivale a flex: 1 1 auto. Los items pueden crecer o reducirse según el caso y flex-basis se calcula según el contenido.
  • flex: none Equivale a flex: 0 0 auto. Los items no crecen ni se reducen y flex-basis se calcula según el contenido.

Tabla de combinación de propiedades (flex)

Propiedad Ejemplo Descripción
flex: auto; flex: 1 1 auto; El elemento puede crecer y reducirse, tamaño base según el contenido.
flex: none; flex: 0 0 auto; El elemento no crece ni se reduce, tamaño base según el contenido.
flex: 2 1 100px; flex-grow: 2;
flex-shrink: 1;
flex-basis: 100px; 		Crece el doble que otros, se reduce si es necesario, tamaño base 100px.
flex: 0 1 150px; flex-grow: 0;
flex-shrink: 1;
flex-basis: 150px; 		No crece, se reduce si es necesario, tamaño base 150px.
