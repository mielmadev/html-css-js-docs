Cursors en CSS

Cursor

Si necesitamos modificar el cursor estándar por uno personalizado, bien a nivel global de página o en ciertos elementos (formularios, enlaces...), existe la propiedad cursor para ello.

Esta propiedad tiene una larga lista de valores estándar y éstos utilizarán los iconos definidos a nivel del Sistema Operativo. 


abbr{
	cursor: help;
}

En algunos desarrollos aún se puede encontrar, bien por herencia o bien por desconocimiento, el valor hand. Este valor nunca ha sido estándar, sólo funcionaba en algunos IE, simulando el valor pointer.

Cursor Personalizado

También podemos añadir nuestro propio cursor personalizado, que será una imagen: 


.bloque{
	cursor: url(ruta/archivo.ext), auto;
}
personalizado

Si necesitamos ajustar las coordenadas del cursor, en el eje X e Y, se añaden en la declaración (similar a como ajustaríamos una imagen de fondo): 


.bloque{
	cursor: url(ruta/archivo.ext) 15 15, auto;
}
personalizado

Hay que tener mucho cuidado con estos cambios porque los usuarios estamos acostumbrados a unos cursores estándar. Si resulta complicado apuntar a los elementos con el cursor personalizado y/o no se entiende el propósito del mismo, va a tener un efecto disuasorio.
