22. Transformaciones

Transformaciones 2D

Podemos usar diferentes transformaciones sobre un elemento (bloque y/o texto), como la rotación, la translación (desplazamiento), el escalado y el sesgado (retorcimiento).

La sintaxis base es la siguiente:

transform: tipoTransformacion(valor);

transform: rotate(45deg); ejemplo transformación de rotación de 45°

En función del tipo de transformación que vayamos a utilizar, aplicaremos diferentes valores de configuración:

  • Rotación: rotate(deg); - n grados
  • Translación: translate(x, y); - ejes X-Y
  • Escalado: scale(x, y); - ejes X-Y
  • Sesgado: skew(deg, deg); - grados ejes X-Y

En el siguiente ejemplo, posiciona el cursor encima de cada uno de los 4 bloques para que se aprecie mejor la transformación que se ha realizado:

 transform: rotate(45deg);

 transform: translate(20px, 20px);

 transform: scale(1.3, 1.3);

Otros valores y particularidades

Si utilizamos un valor en % para la traslación, entonces el % hace referencia al tamaño del propio elemento (no a la anchura/altura del contenedor padre). Es especialmente útil cuando el elemento no tiene un tamaño fijo:

Transformación Visual Código
translate(100%, 0)
A
 transform: translate(100%,0);
translate(-100%, 0)
B
 transform: translate(-100%,0);
translate(0, 100%)
C
 transform: translate(0,100%);
translate(0, -100%)
D
 transform: translate(0,-100%);

En la rotación podemos sustituir los deg por turn, indicando el número de giros que queramos que tenga el elemento. Por lo tanto 1turn = 360deg:

Transformación Visual Código
360deg
A

1turn

3turn
C
 transform: rotate(3turn);

Transformaciones en elementos padre e hijo

Podemos tener elementos padre con una transformación y luego elementos hijo que se rebelan y tienen su propia transformación, inversa a la del padre:

Botón Visual Código
Botón 1
.btn-demo-1 { transform: skew(-40deg); }
/* hover: .btn-demo-1:hover { transform: skew(0deg); } */
Botón 2
.btn-demo-2 { transform: skew(-40deg); }
.btn-demo-2-input { transform: skew(40deg); }
/* hover: .btn-demo-2:hover { transform: skew(0deg); }
.btn-demo-2:hover .btn-demo-2-input { transform: skew(0deg); } */
Botón 3 .btn-demo-3 { transform: skew(-40deg); }
.btn-demo-3-span { transform: skew(40deg); }
/* hover: .btn-demo-3:hover { transform: skew(0deg); }
.btn-demo-3:hover .btn-demo-3-span { transform: skew(0deg); } */

En este ejemplo, dentro de un elemento button, si queremos que a los elementos hijo se les aplique correctamente la transformación, será necesario añadirles alguna propiedad extra que aún no hemos visto (se puede utilizar la propiedad display con los valores block o inline-block para que así aplique la transformación).

Esto es porque las transformaciones sólo se aplican de forma directa en elementos que tienen de manera nativa la propiedad display con el valor block. Más adelante explicaremos esta propiedad.

Combinación de transformaciones

Es posible combinar varias (o todas) transformaciones a la vez. Se realizarán en el orden que se haya indicado:

Transformaciones combinadas Visual Código
rotate + translate + scale
A
 transform: rotate(45deg) translate(50px,20px) scale(1.3, 1.3);
rotate + transform-origin: left top
B
 transform: rotate(70deg);
transform-origin: left top;
rotate + transform-origin: 20% 20%
C
 transform: rotate(-70deg);
transform-origin: 20% 20%;
rotate + transform-origin: right bottom
D
 transform: rotate(70deg);
transform-origin: right bottom;

Este punto de origen se aplica para todas las transformaciones, no sólo al rotado.

Al pasar el cursor por encima de este ejemplo, podemos ver la perspectiva, cómo gira sobre su eje, manteniendo el aspecto 3D:

Matrix

En realidad, los nombres de las propiedades que utilizamos para hacer las transformaciones equivalen a una representación matemática de la transformación en sí misma, por lo que podríamos hacer cualquier tipo de 

      transform: rotate(45deg) es equivalente a:

      transform: matrix(0.707107, 0.707107, -0.707107, 0.707107, 0, 0);

Creación de Formas con transformaciones

Estas transformaciones nos permiten simular imágenes sencillas y no tan sencillas con código CSS, por ejemplo, un triángulo:

HTML:
<div class="triangulo"></div>

CSS:
.triangulo{
width: 0;
height: 0;
border-bottom: 140px solid #cbebff;
border-left: 70px solid transparent;
border-right: 70px solid transparent;
}

Hemos creado un triángulo base y luego en el resto simplemente se realiza una rotación de 90, 180 y 270 grados para utilizar el triángulo en la dirección que queramos.
