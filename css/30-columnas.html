30. Columnas

¿Qué son las columnas?

Podemos distribuir un texto en columnas, de tal manera que el texto fluya en función de la cantidad de texto, las dimensiones del bloque, todo de manera automática

Esta propiedad no está pensada para maquetar bloques de contenido generales (cabecera / menú, por ejemplo), sino que debe utilizarse para distribuir textos, o imágenes con textos.

La sintaxis consta de las siguientes propiedades obligatorias:

  • column-count: Nº de columnas
  • column-width: Anchura de las columnas (valor auto o anchura)

Estas dos propiedades se pueden unificar en la siguiente propiedades: 

      column: 3;         /* 3 columnas de anchura automática */
      column: 3 auto;    /* 3 columnas de anchura automática */
      column: 3 250px;   /* 3 columnas de 250px cada una */

Y las siguiente propiedades opcionales: 

    column-gap: Separación entre columnas
    column-rule: Línea entre columnas (misma sintaxis que un borde)
    column-fill: Balanceo entre el contenido de las columnas (valores: auto o balance)
