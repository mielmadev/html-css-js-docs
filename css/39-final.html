Para acabar

Una misma web, cientos de diseños diferentes: CSS Zen Garden

El proyecto, creado por Dave Shea, consiste en que cualquier desarrollador pueda implementar una CSS propia para maquetar los contenidos de un archivo HTML. Sólo existe una única condición: el código HTML no se puede modificar ya que todos los diseños utilizan el mismo archivo HTML para visualizarse. Cada desarrollador envía su hoja de estilos y a partir de ahí, se juntan los dos archivos.

Las imágenes, por lo tanto, tendrán que estar como fondo y alojadas en el servidor que indique el desarrollador.

Hay diseños muy creativos, en especial, el comic y el cine.
