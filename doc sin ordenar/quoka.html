¿Qué es Quokka.js?

Quokka.js es una extensión para Visual Studio Code que te permite ejecutar y ver resultados de código JavaScript o TypeScript en tiempo real, directamente en el editor. Muestra los valores de las variables, resultados de expresiones y salidas de consola al lado de tu código, facilitando la experimentación, depuración y aprendizaje rápido sin necesidad de configurar un entorno o archivo adicional.

Ideal para pruebas rápidas, aprendizaje interactivo y depuración eficiente.

¿Cómo usar Quokka.js en Visual Studio Code?

  1. Instala la extensión Quokka.js desde el marketplace de VS Code.
  2. Abre o crea un archivo JavaScript o TypeScript.
  3. Haz clic derecho en el editor y selecciona "Start Quokka.js" o usa el comando (Ctrl+Shift+P) y escribe "Quokka: Start on Current File".
  4. Escribe tu código. Verás los resultados y valores de las variables en tiempo real, directamente al lado de tu código.
  5. Puedes probar expresiones, ver resultados de funciones y depurar rápidamente.

Ejemplo:

const a = 2 + 3; // ← Quokka mostrará: 5
console.log(a);   // ← Quokka mostrará: 5

Quokka es ideal para pruebas rápidas y aprendizaje interactivo de JavaScript/TypeScript.
