34. Grid (parte 2)

Areas

Grid nos permite ubicar los elementos con un sistema alternativo al de las líneas: definiendo áreas.

Primero vamos a establecer las filas y columnas que va a tener nuestro elemento contenedor, como hemos hecho hasta ahora:

HTML:
<div class="grid-container">
<div class="item item-cabecera"><p>cabecera</p></div>
<div class="item item-contenido"><p>contenido<br>principal</p></div>
<div class="item item-publi"><p>publicidad</p></div>
<div class="item item-pie"><p>pie</p></div>
</div>
CSS:
Padre
.grid-container{		
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(3, auto);
}
hijos
.item{	
  padding: 20px;
}

Después vamos a poner el nombre de un área a cada elemento hijo mediante la propiedad grid-area. El nombre puede ser cualquier cadena de texto, sin espacios ni caracteres especiales. No hay que poner tampoco comillas a los nombres. Lo recomendable es que pongamos un nombre que nos recuerde fácilmente al elemento que estamos nombrando:

Hijos
.item-cabecera{
  grid-area: cabecera;
}
.item-contenido{
  grid-area: contenido;
}
.item-publi{
  grid-area: publi;
}
.item-pie{
  grid-area: pie;
}

/* Hijos */ .item-cabecera{ grid-area: cabecera;} .item-contenido{ grid-area: contenido;} .item-publi{ grid-area: publi;} .item-pie{ grid-area: pie;}

Padre
.grid-container{
	display: grid;
	grid-template-columns: repeat(3, 1fr);
	grid-template-rows: repeat(3, auto);
	grid-template-areas:
		"cabecera cabecera cabecera"
		"contenido contenido publi"
		"pie pie pie";
}

Con este método, nos olvidamos de número de líneas y basta con cambiar el dibujo para obtener todas las variaciones que queramos.

CSS sólo puede implementar los dibujos de áreas que se podrían realizar con el sistema de líneas. Si hacemos un dibujo imposible, no se implementará.

alineaciones en el padre

Desde el padre podemos utilizar las siguientes alineaciones:

Alinea las columnas con respecto al contenedor padre.
justify-content
Alinea las filas con respecto al contenedor padre.
align-content
Alinea los elementos horizontalmente con respecto a la columna.
justify-items
Alinea los elementos verticalmente con respecto a la fila.
align-items
[justify-content] [align-content]
place-content:

En una propiedad podemos indicar los valores de las dos propiedades anteriores

Para apreciar mejor todas las propiedades, es recomendable utilizar el inspector y poner visibles las líneas grid:

align-content

justify-items

align-items

notas de area

A los hijos le ponemos lo apodos, y asi sabemos donde irá cada uno. en el padre ponemos grid-template-ares: y luego el esquema que queremos, dibujando con los apodos.

Las areas que queremos que queden vacías se pone . los que necesitemos

Declaración en los hijos:
header { grid-area: h}
main { grid-area: m}
aside { grid-area: a}
      
declaración en el padre:
grid-template-areas:
"h h h"
"m a ."
header (h)
main (m)
aside (a)
