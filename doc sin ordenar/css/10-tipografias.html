10. Tipografías

Usar tipografía

Para aplicar una tipografía en un elemento, utilizaremos la propiedad font-family y como valor, el nombre de la tipografía elegida.

font-family: Arial;

Tenemos que tener en cuenta una serie de particularidades a la hora de declarar la tipografía:

Tipografía segura

Si utilizamos una tipografía que sabemos/suponemos que está instalada en todos los sistemas operativos, no tendremos que hacer nada más. Para tipografías personalizadas, veremos más adelante cómo se pueden utilizar.

Tipografía de sustitución

Si un usuario no dispone de la tipografía que hemos indicado, el navegador la sustituirá por otra (que él decida). En la declaración de la tipografía podemos indicar una o varias tipografías específicas de sustitución (para que el navegador no ponga la que quiera):

font-family: Arial, Verdana, Helvetica;

Serif y Sans Serif

Las tipografías se clasifican en diferentes tipos, siendo las categorías más comunes la Serif (con serifa) y la Sans Serif (sin serifa).

Las tipografías Serif cuentan con una serie de detalles en los extremos de las letras y unas florituras, denominadas serifas. Esto hace que se puedan diferenciar mejor ciertas letras y por esto mismo es la categoría de fuentes usada por excelencia en medios impresos. Sin embargo, en pantalla, estas tipografías no se visualizan tan legibles, entre otros aspectos porque es complicado renderizar tanto detalle en un tamaño pequeño.

Es por ello que en pantalla es más legible utilizar tipografías Sans Serif, que son minimalistas, sin detalles. Por este motivo, lo normal es que después de la lista de tipografías de sustitución añadamos la clasificación de esas tipografías, para asegurarnos que el navegador no nos va a poner una tipografía Serif si nuestro diseño usa Sans Serif (y viceversa):

font-family: Arial, Verdana, Helvetica, sans-serif;
font-family: 'Times New Roman', 'Times MS', serif;

Existen otras categorías de tipografía, como las Monospace (monoespaciadas), donde todas las letras tienen la misma anchura fija y mismo espaciado.

Tipografías Personalizadas

Podemos utilizar cualquier tipografía en nuestras páginas (aunque el usuario no la tenga instalada en su dispositivo).

Para ello, primero tendremos que declarar la tipografía que vamos a utilizar, indicar dónde se encuentra (ruta) y el formato de dicha fuente.

Cada navegador tiene soporte para diferentes formatos, por lo que primero haremos una declaración de tipografía, donde indicaremos la ruta a los archivos de la misma, en diferentes formatos:

Ejemplo con:

  • tipografía de nombre [Terminator]
  • archivos de tipografía llamados [webfont.extension]
  • ubicados en una carpeta llamada [fonts]

@font-face {
  font-family: 'Terminator';
  src: url('fonts/webfont.eot'); /* IE9 Compat Modes */
  src:
    url('fonts/webfont.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('fonts/webfont.woff') format('woff'), /* Modern Browsers */
    url('fonts/webfont.ttf') format('truetype'), /* Safari, Android, iOS */
    url('fonts/webfont.svg#svgFontName') format('svg'); /* Legacy iOS */
}

Como podemos prescindir de las líneas de los IE antiguos, el código actual sería:


@font-face {
  font-family: 'Terminator';
  src:
    url('fonts/webfont.woff') format('woff'), /* Modern Browsers */
    url('fonts/webfont.ttf') format('truetype'), /* Safari, Android, iOS */
    url('fonts/webfont.svg#svgFontName') format('svg'); /* Legacy iOS */
}

Y después, ya podremos utilizarla en cualquier elemento (al igual que hemos hecho con las tipografías comunes): 


h1 {
  font-family: 'Terminator', Helvetica, Sans-Serif;
}

Si disponemos de la tipografía en un único formato y la queremos transformar en el resto de formatos, podemos usar un generador automático, siempre y cuando la licencia de dicha tipografía nos permita transformarla y, sobre todo, distribuirla.

Repositorio de Fuentes

  • Google Fonts: gratuito
  • Adobe Fonts: (antes Adobe Typekit) gratuito si se dispone de una cuenta de Adobe

Cómo usar Google Fonts

Para utilizar una fuente de Google Fonts, sigue estos pasos:

  1. Entra en Google Fonts y selecciona la fuente que deseas.
  2. Copia el enlace que te proporciona Google Fonts y pégalo dentro de la etiqueta <head> de tu documento HTML.
  3. En tu archivo CSS, utiliza la propiedad font-family con el nombre de la fuente seleccionada.

Ejemplo para la fuente Roboto:

<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
body {
          font-family: 'Roboto', sans-serif;
        }

Iconografía creada con Tipografía

Podemos crear nuestra propia tipografía personalizada pero, en vez de letras y números, contendrá iconos que podremos utilizar a modo de imagen, con la ventaja que podremos cambiar el tamaño, color y otros estilos de forma directa (al fin y al cabo es un texto).

Estas tipografías las podemos crear manualmente o utilizar un generador automático (gratuito) como IcoMoon.
