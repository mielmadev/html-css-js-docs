25. Dimensiones

Anchura

Si queremos establecer una anchura fija a un elemento, utilizaremos la propiedad width y la medida expresada en la unidad que queramos (pixels, %, rem, vw...):

width: 250px;

Buenas prácticas: Por regla general, evita usar px en elementos de bloque, ya que no se adaptan bien al diseño responsive. Es mejor usar % para que el elemento ajuste su tamaño según el contenedor padre. Los elementos de bloque ocupan el 100% del ancho de su contenedor por defecto y saltan de línea automáticamente.

Para limitar el tamaño de los elementos y mejorar la adaptabilidad, utiliza min-width y max-width:


width: 80%;
max-width: 900px;
min-width: 300px;

Así el elemento será flexible, pero nunca será menor de 300px ni mayor de 900px, facilitando el diseño responsive y evitando problemas de visualización en pantallas muy grandes o pequeñas.

Sólo estableceremos una anchura concreta en aquellos elementos que lo necesiten, por motivos de maquetación.

También es posible utilizar una anchura mínima y una anchura máxima, que nos será muy útil en RWD:


min-width: 200px; /* anchura mínima de 200px */
max-width: 700px; /* anchura máxima de 700px */

Centrado horizontal del bloque

Cuando un elemento tiene una anchura menor que el 100% podemos centrarlo en el eje-X con varias técnicas. La más sencilla es utilizar márgenes laterales simétricos y automáticos.


p {
  width: 50%;
  margin: 0 auto;
}

width:50%;margin:0 auto;

Para centrar el texto dentro del bloque, añade text-align: center;:


p {
  width: 50%;
  margin: 0 auto;
  text-align: center;
}

Así el bloque y su contenido estarán centrados.

Altura

Si queremos establecer una altura fija a un elemento, utilizaremos la propiedad height y la medida expresada en la unidad que queramos:

height: 150px;

Generalmente solo se aplica una altura fija a elementos cuyo contenido no sea fluido, para evitar que el contenido se trunque y no se visualice.

Al igual que ocurre con width, podemos establecer una altura mínima y máxima:


min-height: 50px; /* altura mínima de 50px */
max-height: 200px; /* altura máxima de 200px */

Desbordamiento

Cuando establecemos una anchura y/o altura fija a un elemento y su contenido ocupa más espacio, se produce un desbordamiento de contenido y sobresale del elemento.

En ese momento se decide si esto es fallo nuestro de maquetación y el elemento no necesita unas dimensiones fijas, o bien lo hemos hecho a propósito porque sí queremos que el contenido se desborde. En este caso, tenemos una propiedad overflow para controlar la visibilidad o no del contenido que se desborda:

  • visible: Muestra el contenido que sobresale (comportamiento por defecto)
  • hidden: Oculta el contenido que sobresale
  • scroll: Aparecen barras de scroll
  • auto: Aparecen barras de scroll en caso de que haya desbordamiento
overflow: hidden;

También tenemos unas propiedades específicas para indicar el desbordamiento en cada eje (X,Y) de forma concreta. Los valores de estas propiedades son los mismos que para overflow:


overflow-x: valor;
overflow-y: valor;
Valores de la propiedad overflow
Valor Descripción Comportamiento
visible Contenido desbordado se muestra Por defecto, el contenido sobresale del contenedor
hidden Oculta el contenido desbordado El contenido fuera del contenedor no se muestra
scroll Añade barras de desplazamiento Siempre aparecen barras de scroll, aunque no haya desbordamiento
auto Barras de desplazamiento automáticas Las barras de scroll aparecen solo si hay desbordamiento

URL o Palabras muy largas

En ocasiones tenemos palabras o URLs muy largas que no entran dentro de un elemento. Podemos hacer que la palabra se divida en varias partes para evitar el desbordamiento con la propiedad word-wrap (ahora recomendada como overflow-wrap).

overflow-wrap: break-word;

Función calc()

La función calc() permite realizar cálculos matemáticos automáticamente, con la posibilidad de mezclar diferentes unidades de medida.

No es una propiedad, pero se utiliza en la declaración de valores de una propiedad:


.bloque {
  width: calc(50% - 90px);
}
