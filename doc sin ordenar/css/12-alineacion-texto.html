12. Alineación de texto

Introducción

Nota: estas alineaciones hacen referencia a la alineación de textos dentro de su elemento contenedor. Si queremos alinear bloques de contenido se utilizan otras propiedades (en este caso, de maquetación).

Alineación horizontal

Por defecto, el navegador alinea los textos desde el lazo izquierdo, salvo dos excepciones: los elementos th y caption de las tablas.

Por motivos de legibilidad y accesibilidad, no se debería de utilizar la alineación justificada en la web, mientras los navegadores no puedan implementar técnicamente esta alineación de manera adecuada.

La interdisciplinariedad impulsa la innovación tecnológica mediante la colaboración activa entre expertos de disciplinas diversas, integrando conocimientos especializados para abordar problemas complejos de manera eficiente. Este enfoque promueve soluciones creativas, fomenta el pensamiento crítico, estimula el intercambio de ideas, y fortalece la capacidad de las organizaciones para adaptarse a entornos dinámicos, logrando resultados sostenibles y transformadores.

Está en proceso de desarrollo una nueva propiedad text-justify que permitirá ajustar mejor la alineación justificada (aunque no será perfecto y seguirá dando problemas de accesibilidad), pero todavía falta mucho para poder probarla.

Alineación vertical

La alineación vertical se puede establecer entre textos, imágenes y contenidos de celdas de tabla.

Ajuste de texto

Podemos controlar las palabras únicas (conocidas como viudas y huérfanas) que se encuentran solas en la última línea de un elemento.

Las viudas son palabras solas en la parte superior del elemento y las huérfanas están solas al final del elemento.

A algunas personas les dificulta la lectura del contenido por lo que se suele recomendar evitarlos. Hasta hace muy poco era imposible técnicamente, pero ahora disponemos de la propiedad text-wrap para ello.

text-wrap: auto wrap nowrap balance pretty stable

El fantasma de la ópera

  • auto o wrap:
    El texto se ajusta a la siguiente línea según el ancho del padre.
  • nowrap:
    El texto permanece en una sola línea, desbordando el padre si es necesario.
  • balance:
    Equilibra las líneas para longitudes similares (si el navegador lo soporta).
  • pretty:
    Evita palabras huérfanas en la última línea (si el navegador lo soporta).
  • stable:
    Mantiene la estabilidad del ajuste al editar texto (si el navegador lo soporta).
