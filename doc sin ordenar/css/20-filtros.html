20. Filtros CSS

Los filtros nos permiten manipular tanto elementos html como imágenes en varios aspectos. La sintaxis base es la siguiente:

filter: nombrefiltro(valor);
filter: nombrefiltro(valor);

filter: blur(20px); /* ejemplo filtro de desenfoque */

En función del tipo de filtro que usemos, tendremos diferentes valores de configuración:

Filtro Para qué es Unidades
blur() Desenfoca el elemento px
brightness() Ajusta el brillo número (0 a ∞, 1=normal)
saturate() Aumenta o reduce la saturación % o número (1=normal)
hue-rotate() Rota el tono de color deg
contrast() Ajusta el contraste % o número (1=normal)
invert() Invierte los colores %
grayscale() Convierte a escala de grises %
sepia() Aplica efecto sepia %
opacity() Ajusta la opacidad %
drop-shadow() Agrega sombra al elemento offset-x, offset-y, blur-radius, color

Podemos combinar varios filtros a la vez:

filter: grayscale(100%) sepia(100%);

Y existe la posibilidad de crear filtros personalizados con SVG:

HTML:
        <svg>
  <filter id="filtroperso">
    <feTurbulence type="turbulence" baseFrequency="0.01 0.02" numOctaves="1" result="noise">
    </feTurbulence>
    <feDisplacementMap in="SourceGraphic" in2="noise" scale="50">
    </feDisplacementMap>
  </filter>
</svg>

<img src="ruta/imagen.ext" alt="Imagen con Filtro SVG">

CSS:
img{
filter:url(#filtroperso);
}
