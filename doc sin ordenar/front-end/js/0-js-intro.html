¿Qué es JavaScript?

JavaScript (abreviado comúnmente JS) es un lenguaje de programación interpretado (no compilado), dialecto del estándar ECMAScript.

Fue creado por Brendan Eich en 10 días, como parte de su trabajo en Netscape. Desde entonces ha ido evolucionando.

Nota: JavaScript fue creado en 1995 y su evolución ha sido clave para el desarrollo web moderno.

Diferencias entre JavaScript y Java

JavaScript y Java NO son lo mismo. Son lenguajes de programación completamente diferentes, simplemente tienen las 4 primeras letras del nombre en común.

No digas Java si en realidad quieres decir Javascript, o puedes tener problemas (apuntarte a una oferta de trabajo errónea, que tus compañeros de trabajo te miren muy mal o dejen de hablarte por equivocar conceptos... -esto es verídico-). Si quieres abreviar el nombre, usa el término correcto: JS.

JavaScript en el desarrollo web

Javascript se utiliza principalmente del lado del cliente, implementado como parte de un navegador web para añadir mejoras en la interfaz de usuario y páginas web dinámicas. Con HTML hacemos la estructura semántica, con CSS la presentación visual y por último, con JS añadimos la interacción.

Todos los navegadores modernos interpretan el código JavaScript integrado en las páginas web. Para interactuar con una página web se provee al lenguaje JavaScript de una implementación del Document Object Model (DOM). Javascript es el único lenguaje de programación que entienden de forma nativa los navegadores.
