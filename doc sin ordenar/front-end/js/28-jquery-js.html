Notas sobre jQuery (MielmaDev)

jQuery es una biblioteca de JavaScript rápida, pequeña y rica en funciones. Simplifica tareas comunes de JavaScript como la manipulación del DOM, el manejo de eventos, la animación y las solicitudes AJAX.

¿Cuándo usar jQuery hoy en día? Aunque jQuery fue muy popular, hoy la mayoría de sus funciones ya existen en JavaScript moderno. Sin embargo, sigue siendo útil si trabajas en proyectos antiguos o necesitas compatibilidad con navegadores antiguos.

Ventajas principales:

  • Sintaxis sencilla para seleccionar y modificar elementos del DOM.
  • Facilita el manejo de eventos y animaciones.
  • Permite hacer peticiones AJAX de forma simple.

Desventajas: Añade peso extra a la web y no es necesario en proyectos nuevos con JavaScript moderno.

¿Cómo se usa? Solo tienes que incluir la librería en tu HTML y puedes empezar a usar la sintaxis $(selector).

¿Por qué usar jQuery?

  • Simplifica la sintaxis de JavaScript.
  • Gestiona automáticamente las diferencias entre navegadores.
  • Proporciona una amplia gama de funciones útiles.

Ejemplo básico de jQuery

// Asegurarse de que el DOM esté listo
$(document).ready(function() {
  // Código jQuery aquí
});

Por ejemplo, para ocultar un elemento con id caja al hacer clic en un botón:

$('#boton').click(function() {
  $('#caja').hide();
});
