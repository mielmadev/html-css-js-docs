Normativas de Accesibilidad

Existen varias normativas y estándares internacionales que guían la accesibilidad web. Las Pautas de Accesibilidad para el Contenido Web (WCAG) son las más reconocidas a nivel mundial y proporcionan recomendaciones para hacer que el contenido web sea más accesible para personas con discapacidades. La versión más reciente, WCAG 2.1, incluye criterios específicos para mejorar la accesibilidad en áreas como el texto alternativo, la navegación por teclado y el contraste de colores.

Además de las WCAG, otros estándares como la Sección 508 en Estados Unidos y la EN 301 549 en Europa establecen requisitos específicos para la accesibilidad de productos y servicios digitales.

Es fundamental que los desarrolladores web conozcan y apliquen estas normativas para garantizar que sus sitios sean inclusivos y accesibles para todos los usuarios.

  • WCAG 2.2: Pautas de Accesibilidad para el Contenido Web, estándar internacional para la accesibilidad web.
  • ATAG 2.0: Pautas de Accesibilidad para Herramientas de Autor, orientadas a la accesibilidad en editores y CMS.
  • UNE 139803 (Una Norma Española): Norma española que adapta las WCAG al contexto nacional.
  • ISO/IEC 40500: Estándar internacional que adopta las WCAG 2.0 como norma ISO.
  • EN 301 549: Norma europea para la accesibilidad de productos y servicios TIC.
  • WAI-ARIA: Especificación para mejorar la accesibilidad de aplicaciones web dinámicas.
  • WAI-EPUB: Directrices para la accesibilidad en publicaciones electrónicas (EPUB).
  • ISO/IEC 23761:2021: Estándar internacional para la accesibilidad de productos y servicios TIC.
  • ISO 14283-1:2014: Norma relacionada con la accesibilidad, especialmente en el ámbito de la salud y la geriatría.
